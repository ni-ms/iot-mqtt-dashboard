# IoT MQTT Dashboard

A customizable dashboard for monitoring and managing IoT devices via MQTT. This application allows users to visualize real-time data from MQTT brokers through a flexible and configurable interface.

## Getting Started

To run the application in development mode, execute the following command:

```bash
npm start
```

## Features

- **Real-time Monitoring**: Connect to MQTT brokers and monitor data streams in real-time.
- **Customizable Dashboard**: Configure layouts and cards to suit specific monitoring needs.
- **Protocol Support**: Robust integration with MQTT for IoT device communication.
- **Theming**: Support for multiple themes to improve readability and user preference.

## Technical Overview

This project is built using React Native and Expo, utilizing MobX for state management and an organized service-oriented architecture for MQTT handling.
